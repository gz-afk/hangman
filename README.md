#  Hangman Game

A simple and interactive **Hangman game** built using **JavaScript, Python (Flask), HTML, and CSS**.
Guess the hidden word before the stick figure is fully drawn!

---

##  Features

*  Classic Hangman gameplay
*  Random word selection from a customizable word list
*  Clean UI with CSS styling
*  Real-time updates using JavaScript
*  Backend powered by Flask (`app.py`)

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)

##

---

##  Installation & Setup

1. **Clone the repository**

```
git clone https://github.com/your-username/hangman-game.git
cd hangman-game
```

2. **Install dependencies**

```
pip install flask
```

3. **Run the app**

```
python app.py
```

4. **Open in browser**

```
http://127.0.0.1:5000/
```

---

##  Customization

* Modify the word list inside `app.py`
* Update styles in `style.css`
* Improve game logic in `script.js`
